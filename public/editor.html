Atom editorini O'rnatish

Go dasturlash tilda programmlar yozish uchun atom editorini yuklab olamiz
Atom
windows uchun avtomatik xolatda o'rnatiladigan bo'lib yuklanadi
agar linux sistemi uchun o'rnatyotgan bo'lsangiz, bu quydagidek:
.deb faylini yuklaysiz.

Linux uchun yuklagan bo'lsangiz quyidagidek o'rnatasiz:
atom yuklangan quttiga borib terminalda
sudo dpkg --instal atom-amd64.deb buyruqini yurgizasiz
Atom haqida farqli informatsiya

editorni o'rnatib bo'lgandan keyin, uni ochib chap tomondan tepada
edit degan referensni ochib uning ichida preferences deganini ochamiz
u bizga yangi oyna ochadi uning ichida chap tomondan Install deganini bosib
go-plus paketini axtaramiz va yuklaymiz u Go bilan ishlashni ossonlashtiradi

agar xozircha editor ishlatishni xoxlamasangiz
Go kodini onlayn yozish ham mimkin
lekin biz sistemga qarashli kod yozishimiz xam mumkin, u web browserda ishlamaydi
